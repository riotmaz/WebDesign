HTML (HyperText Markup Language) is the standard markup language used to create and structure content on the web.
It provides the basic framework for web pages, which browsers interpret to display text, images, links, and other elements.

Key Features of HTML:
1. Markup Language: HTML uses a system of tags to mark up the structure of a web page.
2. Browser Interpretation: Web browsers read HTML files and render them into user-friendly content.
3. Content Structuring: It organizes content into headings, paragraphs, lists, tables, links, images, and more.

Explanation of Tags:
- DOCTYPE html: Declares the document type and version of HTML being used.
- html: The root element that contains all other elements.
- head: Contains metadata like the title, styles, and scripts.
- title: Sets the title of the web page shown in the browser's tab.
- body: Contains the visible content of the web page.
- h1: A heading tag (one of six levels, h1 to h6).
- p: A paragraph tag.
- a: An anchor tag used for creating hyperlinks.



Common HTML Tags:
1. Headings: h1 to h6 for titles and subtitles.
2. Paragraphs: p for blocks of text.
3. Links: a for hyperlinks.
4. Images: img for embedding images.
5. Lists:
- Unordered: ul with li for bullet points.
- Ordered: ol with li for numbered lists.
6. Tables: table, tr, td, th for tabular data.
7. Forms: form with input, textarea, button, etc., for user input.



HTML Versions:
- HTML4: Introduced in 1997, widely used for many years.
- XHTML: A stricter version of HTML.
- HTML5: The latest version, introduced in 2014, supports modern web development needs (multimedia, responsive design).

HTML is often paired with:
- CSS (Cascading Style Sheets): For styling and layout.
- JavaScript: For interactivity and dynamic behavior.
